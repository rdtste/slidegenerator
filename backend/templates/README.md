# PowerPoint Templates

Lege hier `.pptx`-Dateien als Folienmaster ab.

Jede `.pptx`-Datei wird automatisch als Template erkannt.
Der Dateiname (ohne Extension) wird zur Template-ID.

## Beispiel

- `corporate.pptx` → Template-ID: `corporate`
- `minimal_dark.pptx` → Template-ID: `minimal_dark`

## Eigene Templates erstellen

1. Öffne PowerPoint
2. Gehe zu **Ansicht → Folienmaster**
3. Definiere Layouts (Titel, Inhalt, Zwei Spalten, etc.)
4. Speichere als `.pptx`
5. Kopiere die Datei in diesen Ordner (oder lade sie über die UI hoch)
